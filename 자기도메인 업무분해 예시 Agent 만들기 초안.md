# 자기도메인 업무분해 예시 Agent 만들기 초안

## [도메인 맥락]

나는 엄마 15년 차이고, 중1~중3 자녀의 국어·수학·영어 내신 시험범위에 맞는 자료찾기와 시험문제 생성 작업을 자동화/정리하고 싶습니다. (예: OO중학교 2학년 1학기 중간고사, 수학 교과서 2단원)

## [네 동작 분석]

1. **분해**: 학년·과목·교과서·시험범위·성취기준·난이도·문항유형으로 구조화
2. **패턴**: 교과서 핵심 개념, 빈출 유형, 학생의 오답 경향을 분석
3. **추상화**: 개인별 맞춤 학습 로드맵과 시험 대비 문제 세트 생성
4. **알고리즘** (도구 체인으로 구성)
   1. Input: `{학년, 과목, 교과서명, 시험범위(단원)}`
   2. Tool A (자료 검색): 시험범위에 해당하는 교과서 목차/성취기준 문서를 조회
      - 출처: 국가교육과정정보센터 성취기준 DB, 교과서 목차 PDF 등 실제 참조 소스를 명시
   3. Tool B (개념 매핑): 검색된 자료에서 핵심 개념 N개, 대응 성취기준 코드를 추출
   4. Tool C (문제 생성): 난이도(상/중/하) 비율을 지정하여 문항 유형별(객관식/서술형) 문제+정답+해설 생성
   5. Output 저장 후, 학생 채점 결과(오답 목록) 입력 시 → Tool D (추천): 취약 개념 재학습 자료 추천

## [출력 요구]

다음 3개 섹션을 포함한 마크다운 문서, 한국어로 작성:

1. **학습지**: 단원별 핵심 개념 요약 (개념당 3줄 이내)
2. **시험 예상문제**: 총 10문항 (객관식 7, 서술형 3), 난이도 상/중/하 = 2/5/3 비율
3. **정답 및 해설**: 문항 번호 - 정답 - 해설(3~5줄) - 관련 성취기준 코드

## [검증 기준]

- (자동 검증) 성취기준 코드 커버리지 100%, 문항 수·난이도 비율 스펙 일치, 오탈자/중복문항 0건
- (인적 검증) 현직 중학교 교사 1인이 "학원이나 학교에서 실제 보조 교재로 사용할 수 있는 수준이다"라고 평가할 것

## [제약사항]

- 교과서 원문을 그대로 복제하지 않는다 (저작권 준수, 개념 재구성 방식으로 작성).
- 근거 없는 성취기준·개념·문제를 생성하지 않는다 (사실확인 없이 지어내는 내용 금지).
- 위험하거나 검증되지 않은 외부 자료는 사용 전 출처를 표기한다.
