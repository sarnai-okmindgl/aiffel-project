# 자기도메인 업무분해 예시 과제

## [도메인 맥락]

나는 엄마 15년 차이고, 아이들 공부범위에 맞는 자료찾기와 시험문제 생성 작업을 자동화/정리하고 싶습니다.

## [네 동작 분석]

1. **분해**: 학년·과목·교과서·시험범위·성취기준·난이도·문항유형으로 구조화
2. **패턴**: 교과서 핵심 개념, 빈출 유형, 학생의 오답 경향을 분석
3. **추상화**: 개인별 맞춤 학습 로드맵과 시험 대비 문제 세트 생성
4. **알고리즘**: 시험범위 입력 → 관련 학습자료 수집 → 핵심 개념 및 성취기준 매핑 → 난이도별 문제·해설 생성 → 학습 결과에 따라 다음 학습 추천

## [출력 요구]

맞춤형 학습지·시험 예상문제·정답 및 해설 형식으로, 한국어로.

## [검증 기준]

현직 중학교 교사 한 사람이 "학원이나 학교에서 실제 보조 교재로 사용할 수 있는 수준이다."라고 평가할 것.
