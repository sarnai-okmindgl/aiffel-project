# aiffel-project

AI Agent를 만드는 과정을 간략히 정리한 저장소입니다.

## AI Agent를 만드는 방법

1. **목표와 역할 정의**
   Agent가 무엇을 해결할지, 어떤 입력을 받고 어떤 결과를 내야 하는지 명확히 정한다.

2. **LLM 선택**
   Agent의 두뇌 역할을 할 언어 모델(Claude 등)을 선택하고, 시스템 프롬프트로 역할과 규칙을 정의한다.

3. **도구(Tools) 연결**
   웹 검색, 파일 읽기/쓰기, API 호출 등 Agent가 실제로 실행할 수 있는 도구를 정의하고 연결한다.

4. **메모리/컨텍스트 관리**
   이전 대화나 작업 결과를 기억하고 활용할 수 있도록 대화 이력, 벡터 검색(RAG) 등을 구성한다.

5. **계획 → 실행 → 관찰 루프**
   Agent가 스스로 다음 행동을 계획하고, 도구를 실행하고, 결과를 관찰해 다시 계획하는 반복 루프를 만든다.

6. **평가와 반복 개선**
   실제 사용 사례로 테스트하며 실패 케이스를 찾아 프롬프트/도구/로직을 개선한다.
